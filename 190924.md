## 백준 17471번 - 게리맨더링

N개의 원소에 대한 부분집합

```
[1, 2, 3]
    A         B
000 []        [1, 2, 3]
001 [3]       [1, 2]-----------------------
010 [2]       [1, 3] 여기까지만 계산하면 된다.
011 [2, 3]    [1] -------------------------
100 [1]       [1, 3]
101 [1, 3]    [2]
110 [1, 2]    [3]
111 [1, 2, 3] []
```

```python
# 공집합은 제외해야 함
for i in range(1, (1<<N)//2):
    for j in range(N):
        if i & (1<<j) != 0:
            A.append(j+1) # j+1을 선거구 번호로 활용
        else:
            B.append(j+1)
```
